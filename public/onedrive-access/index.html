Hướng dẫn xử lý quyền truy cập OneDrive trong Microsoft 365

Vấn đề

Khi một User bị xóa trong tổ chức Microsoft 365 và sau đó được tạo lại với tên trùng với User đã xóa, người dùng này không thể truy cập các file được chia sẻ và nhận được thông báo "Không có quyền truy cập file này".

Cách xử lý

Bước 1: Xử lý tại SharePoint Admin Center

  • Truy cập SharePoint Admin Center
  • Chọn More Features > User Profile
  • Trong phần User Profile, chọn Manage User Profiles
  • Tìm kiếm tài khoản người dùng đang gặp lỗi
  • Chọn Manage Site Collection Owners

Bước 2: Xử lý tại OneDrive cá nhân

  • Truy cập OneDrive cá nhân > OneDrive Settings > Return to Classic OneDrive
  • Thay đổi đường dẫn:
    • Xóa phần: Documents/Forms/All.aspx
    • Thay thế bằng: _layouts/15/people.aspx?MembershipGroupId=0
  • Tìm và xóa User đang gặp lỗi

Lưu ý: Sau khi hoàn thành các bước trên, hãy thử chia sẻ lại file cho User để kiểm tra.
